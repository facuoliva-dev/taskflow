# taskflow

📌 TaskFlow es una aplicación de consola desarrollada en C# que permite gestionar tareas de manera simple.

▶️ Cómo ejecutarlo:
1  Clonar el repositorio
2. Abrir en Visual Studio
3. Ejecutar con F5 o usar:dotnet run

🛠 Tecnologías usadas- C#- .NET- Visual Studio- Git y GitHub
